## How to Access Supplementary Videos

To watch the supplementary videos, please open the "index.html" file located in the supplementary root directory.

## Contents of the "prompts" Folder

The "prompts" folder includes two key files:

1. benchmark_prompts.yaml - This file contains prompts used for experiments involving automatic metrics and user studies.
   
2. qualitative_prompts.yaml - This file includes prompts used for qualitative comparisons and the ablation study.
